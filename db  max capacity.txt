
These are some of the Maximum Capacity Specifications for SQL Server 
(32bit/ 64bit)

	Database size: 524,272 terabytes

	Databases per instance of SQL Server: 32,767
	
	Filegroups per database: 32,767
	
	Files per database: 32,767

	User Connections: 32,767 
	
	File size (data): 16 terabytes
	
	File size (log): 2 terabytes
	
	Columns per Update Statement (Wide Tables): 4096 

	Statistics on non-indexed columns: 30,000

	Columns per Select Statement: 4096
	
	Columns per Insert Statement: 4096
	
	Columns per wide table: 30,000

	Columns per Primary Key: 16
	
	Columns per Foreign Key: 16
	
	Columns per Index Key: 16
	
	Columns per nonwide table: 1024 

	Columns or expressions in a GROUP BY WITH CUBE or WITH ROLLUP 			Statement: 10

	Columns in GROUP BY, ORDER BY: Limited only by number of bytes

	Clustered indexes per table: 01

	XML Indexes: 249 
	
	Partitions per partitioned table or index:
	1,000 (32 bit System), 15,000 (64 bit System)
	
	Parameters per user-defined function: 2,100
	
	Parameters per Stored Procedure: 2,100

	Rows per table: Limited by available storage
	
	Tables per database: Limited by number of objects in a database

	Triggers per table:  Limited by number of objects in a database
